# zrx-link

## Overview

A simple and ultra fast link shortener, powered by [Bun](https://bun.sh/)

Consist of :

- Backend : Made with [Hono](https://hono.dev/)
- Frontend : Made with [Solid](https://solidjs.com/)
- Database : [Redis](https://redis.io/)

## Development

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```
